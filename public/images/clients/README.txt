CARA DOWNLOAD LOGO CLIENT:

1. Buka Google dan cari: "[Nama Brand] logo png transparent"
   Contoh: "BCA logo png transparent"

2. Download logo dengan background TRANSPARAN (format PNG)

3. Simpan dengan nama file sesuai:
   - bca.png
   - mandiri.png
   - telkom.png
   - pertamina.png
   - astra.png

4. Paste semua file logo ke folder ini: public/images/clients/

TIPS:
- Pilih logo resolusi tinggi (minimal 500px)
- Pastikan background transparan
- Format PNG lebih baik dari JPG
- Ukuran file jangan terlalu besar (max 500KB per logo)

SUMBER LOGO RESMI:
